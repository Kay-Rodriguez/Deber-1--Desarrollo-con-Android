Desarrollar una aplicación usando el código base del Taller 1-Android que permita a un agente de tránsito :

#· Registrar accidentes de tránsito por medio de un formulario mismo que debe contener, Tipo de accidente(choque, colisión, atropello), fecha del siniestro, matrícula del auto involucrado, nombre conductor, cedula conductor, observaciones, fotografías y ubicación GPS del lugar donde se sucedió el siniestro.

#· Cada vez que se da click en el botón guardar al terminar el registro de un accidente, el teléfono debe vibrar 5 segundos(consultar la clase Vibrator para implementar esto).

#· Usar componentes de Jetpack Compose

#· Implementar cámara para captura de fotos

#· Obtener ubicación GPS

#· Gestionar permisos dinámicos
